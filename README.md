# Retraction Watch Dataset Analysis

## Project Overview

This project analyzes the **Retraction Watch Database**, a global dataset of retracted scientific papers. The goal is to study patterns in scientific retractions across different countries, journals, and research fields.

The analysis aims to understand:

* Which countries have the highest number of retracted papers
* Common reasons for paper retractions
* Trends in research misconduct
* What lessons India can learn to improve research integrity

This project also provides **policy suggestions for the Government of India** to strengthen research quality and academic ethics.

---

## Dataset

The dataset used in this project is the **Retraction Watch Database**, which contains information about retracted scientific publications.

Key columns in the dataset include:

* Title – Title of the research paper
* Subject – Field of study
* Institution – Author institution
* Journal – Publishing journal
* Publisher – Journal publisher
* Country – Country of the author
* Retraction Date – Date of retraction
* Reason – Reason for retraction

---

## Objectives

1. Perform **exploratory data analysis (EDA)** on retracted research papers.
2. Identify **country-wise trends in retractions**.
3. Study the **most common reasons for retractions**.
4. Analyze **research fields with higher retraction rates**.
5. Provide **policy recommendations for improving research integrity in India**.

## Key Insights

Some important observations from the dataset:

* Certain countries contribute a higher number of retracted papers.
* Common reasons for retraction include:

  * Data fabrication
  * Plagiarism
  * Ethical violations
  * Duplicate publication
* Some research fields show higher vulnerability to misconduct.

---

## Policy Suggestions for India

Based on the analysis, the following measures could help improve research quality in India:

1. **Strengthen research ethics training** in universities.
2. **Mandatory plagiarism detection systems** before publication.
3. **Improve peer-review standards** in journals.
4. **Create a national database for research misconduct cases**.
5. Encourage **open data and reproducible research practices**.
